## CSO/국제개발협력 관심자용

안녕하세요. 외교 공공데이터·AI 경진대회 제출용으로 만든 `K-ODA Compass RAG`를 10분만 테스트해주실 수 있을까요?

이 서비스는 KOICA 사업근거, WDI 개발지표, CPS 국가협력전략 PDF를 AI가 근거로 검색해서 ODA 사업기획서 초안을 만들어주는 도구입니다.

테스트 방법:
1. 데모 링크 접속
2. `AI Builder` 탭 클릭
3. `CSO 탄자니아 공공행정` 시나리오 선택
4. `RAG형 AI 사업기획서 생성` 클릭
5. `근거 Citation`, `브리프`, `원문`을 확인

아래 3가지만 답해주시면 됩니다.
- 추천 이유가 이해되나요?
- CPS/KOICA/WDI 근거 citation이 신뢰에 도움이 되나요?
- 실제 사업 아이디어 기획에 도움이 될까요? 1~5점으로 평가 부탁드립니다.

## 지자체/공공기관 사용자용

안녕하세요. 지자체 국제협력/ODA 사업 발굴에 쓸 수 있는 AI 도구를 테스트 중입니다.

`K-ODA Compass RAG`는 외교·개발협력 공공데이터를 바탕으로 국가·분야 추천, 정책근거, 위험요인, Brief를 자동 생성합니다.

테스트 추천 시나리오:
- `AI Builder` 탭
- `지자체 베트남 디지털정부`
- 결과 생성 후 `Brief MD`와 `Brief PDF` 확인

피드백 질문:
- 지자체 국제협력 사업 발굴 관점에서 유용한가요?
- 어떤 항목이 더 있으면 실제 검토에 도움이 될까요?
- 전반적 유용성 1~5점은 몇 점인가요?

## 기업/스타트업 사용자용

안녕하세요. 개발협력/ODA 파트너십 기회를 찾는 기업용 AI 도구를 테스트 중입니다.

이 도구는 국가별 ODA 기회점수, CPS 정책근거, KOICA 기존 사업, WDI 개발수요를 결합해 사업기획서 초안과 Evidence Pack을 생성합니다.

테스트 추천 시나리오:
- `AI Builder` 탭
- `기업 르완다 ICT·에너지`
- `RAG형 AI 사업기획서 생성`

피드백 질문:
- 기업 입장에서 시장/협력기회 탐색에 도움이 되나요?
- citation 기반 근거가 의사결정에 도움이 되나요?
- 실제로 쓰려면 어떤 기능이 더 필요할까요?
