# User Validation Kit

Use this before submission to collect quick evidence that the service solves a real workflow problem.

## Target reviewers

- 1 CSO/NGO worker or international-development student
- 1 public-sector/local-government user
- 1 company/startup or policy/planning user

## 10-minute test

1. Open the deployed Streamlit URL.
2. Go to `AI Builder`.
3. Pick one scenario.
4. Generate the RAG proposal.
5. Review `근거 Citation`, `브리프`, and `원문`.
6. Answer the questions below.

## Feedback questions

1. Can you understand why this country-sector pair was recommended?
2. Are the CPS/KOICA/WDI citations useful for trust?
3. Would the 1-page brief help you start an ODA project idea?
4. What information is still missing before real project planning?
5. Score usefulness from 1 to 5.

## Evidence to capture

- Reviewer role
- Scenario tested
- Usefulness score
- One positive quote
- One improvement quote
