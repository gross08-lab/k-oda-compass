# Final Submission Checklist

## Must have

- [ ] Streamlit Cloud URL works on a clean browser
- [ ] GitHub repository is public or accessible to judges
- [ ] README has live URL and GitHub URL
- [ ] `AI Builder` produces a proposal without an API key
- [ ] `LLM RAG` works when `OPENAI_API_KEY` is configured
- [ ] `AI검증` tab shows RAG corpus and reproducibility
- [ ] `심사모드` tab is ready for Q&A
- [ ] Presentation deck includes QR codes
- [ ] Three sample outputs are in `sample_outputs/`

## Nice to have

- [ ] Three quick user feedback quotes
- [ ] OCR coverage report included for image-only CPS PDFs
- [ ] OCR refresh rerun if Tesseract Korean OCR is available
- [ ] 2-minute screen recording
- [ ] One-page PDF handout

## Demo fallback

If internet or OpenAI API fails, use local RAG mode and pre-generated sample outputs.
