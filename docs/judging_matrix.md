# Judging Matrix

## Position

K-ODA Compass RAG is not a generic dashboard. It is an evidence-grounded AI policy copilot that converts diplomatic and development-cooperation public data into country ranking, sector recommendation, CPS/KOICA/WDI evidence retrieval, and proposal-grade outputs.

## Criteria Response

| 평가축 | 구현 내용 | 심사 포인트 |
|---|---|---|
| 공공데이터 활용성 | KOICA 2019-2024, WDI 2019-2025, CPS PDF, 정책·리스크 proxy 결합 | 복수 외교·개발협력 공공데이터 융합 |
| AI 혁신성 | RAG 검색, Evidence Pack, 선택적 LLM 생성, 로컬 fallback | 단순 챗봇이 아닌 근거 고정형 생성 |
| 서비스 완성도 | Streamlit MVP, 국가순위, 프로필, 분야추천, AI Builder, 검증, 배포 탭 | 심사장에서 바로 조작 가능 |
| 검증 가능성 | 점수 재현 테스트, RAG 코퍼스 통계, CPS OCR 커버리지, 민감도 분석, citation | 설명 가능한 AI |
| 활용 가능성 | CSO, 지자체, 기업, 정책담당자 시나리오 | 사용자가 바로 사업기획서/브리프 획득 |

## One-sentence Differentiator

K-ODA Compass RAG is a public-data AI planning workflow that cites CPS policy documents, KOICA project evidence, WDI development indicators, and risk proxies in every generated ODA proposal.
